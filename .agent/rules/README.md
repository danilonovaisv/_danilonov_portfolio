# Agent Rules

Este diretório contém as regras e diretrizes para o agente AI seguir durante o desenvolvimento do projeto.

## Estrutura

- Cada arquivo `.md` neste diretório representa um conjunto de regras específicas
- As regras são automaticamente carregadas e aplicadas pelo agente
- Organize as regras por categoria (ex: `coding-standards.md`, `architecture.md`, `testing.md`)

## Regras do Projeto

- `project-context.md` — Fonte de verdade para protótipos e referências visuais em `.context/`
- `context-source-log.md` — Registrar a fonte consultada em `.context/` no resumo da tarefa

## Exemplo de Uso

Crie arquivos markdown com regras específicas:

```markdown
# Coding Standards

- Use TypeScript strict mode
- Follow ESLint rules
- Prefer functional components in React
- Use Tailwind CSS for styling
```
