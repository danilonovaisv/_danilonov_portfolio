# Responsividade Mobile (RWD)

- **viewport meta** é obrigatório
- **mobile-first media queries** devem existir
- Imagens usam **srcset/sizes + lazy loading**
- Touch targets >= 44px
- Tipografia escalável
- Menus com fallback táctil
- Testes automáticos via emuladores e dispositivos reais
