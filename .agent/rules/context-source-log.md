# Registro de Fonte de Pesquisa (.context)

Sempre que usar `.context/` como base para decisões de UI/UX, **registre a fonte consultada** no resumo da tarefa (ou comentário de PR), citando:

- Arquivo(s) consultado(s) em `.context/`
- Seção relevante (quando existir)
- Se é desktop ou mobile

## Exemplo

- Fonte: `.context/PORTFOLIO-PAGE-LAYOUYT.jpg` (desktop)
- Fonte: `.context/PORTFOLIO-PAGE-MOBILE.jpg` (mobile)
- Fonte: `.context/PORTFOLIO-PROTOTIPO-INTERATIVO.md` (Seção "Gallery")
