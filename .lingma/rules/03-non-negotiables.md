# Regras Inegociáveis (Non-Negotiables)

1. **Integridade Visual:** Não invente cores. Use estritamente a paleta `#0048ff`, `#040013`, `#fcffff`.
2. **Zero Lorem Ipsum:** Use os textos reais fornecidos nos documentos `PROTOTIPO INTERATIVO`.
3. **Mobile First Real:** Teste mentalmente em 375px antes de codar.
4. **Sem Scroll Hijacking:** Não sequestre o scroll nativo do navegador, a menos que esteja implementando o sistema de Parallax controlado via container fixo (como especificado no Portfolio Showcase).
5. **Typescript:** Sem `any`. Defina interfaces para Props.
6. **URLs Seguras:** Links externos sempre com `rel="noopener noreferrer"`.
