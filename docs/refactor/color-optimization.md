## Variables Added to `globals.css`

The following variables were added to the `@theme` block:

```css
@theme {
  /* Text Colors */
  --color-text-dark: #111111;
  --color-text-muted: #666666;

  /* Section & Component Backgrounds */
  --color-section-bg: #F4F5F7;
  --color-section-bg-hover: #ECEDF0;
  --color-card-bg: #f7f7f7;

  /* Focus Ring Colors */
  --color-focus-ring: #0057FF;
}
```

## Usage Examples

### Before (Hardcoded)

```tsx
<button className="bg-[#0057FF] text-white hover:shadow-[#0057FF]/40">
  Click me
</button>

<h2 className="text-[#111111]">Title</h2>

<section className="bg-[#F4F5F7]">Content</section>
```

### After (Variables)

```tsx
<button className="bg-primary text-white hover:shadow-primary/40">
  Click me
</button>

<h2 className="text-text-dark">Title</h2>

<section className="bg-section-bg">Content</section>
```
